# Customer-Retention-Churn-Analysis
Customer churn &amp; retention analysis using Power BI
## Customer Retention & Churn Analysis

### Objective
Analyze customer churn behavior and identify key retention drivers in a subscription-based business.

### Tools Used
- Power BI
- Microsoft Excel

### Key Insights
- Month-to-month customers show highest churn
- Early tenure customers are more likely to churn
- Long-term contracts improve retention

### Recommendations
- Promote long-term plans
- Improve early customer onboarding
- Target high-risk churn segments
